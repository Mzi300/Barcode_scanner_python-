# Python Barcode Scanner

## Description
This is a Python-based barcode scanning software.  
It is designed to detect and decode barcodes using a webcam or image input.

## Status
⚠️ Work in Progress  
The project is currently untested and under development.

## Features (Planned)
- Barcode detection using OpenCV
- Barcode decoding using pyzbar
- Support for camera input
- Save scanned results to a file

## Requirements
- Python 3.x
- opencv-python
- pyzbar
